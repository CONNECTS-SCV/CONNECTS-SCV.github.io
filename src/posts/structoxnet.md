## 개요
StructoxNet은 펩타이드(peptide)의 독성(toxicity) 을 3D 구조 기반으로 예측하는 이진 분류 모델입니다. 서열정보와 구조정보를 결합하여 펩타이드의 독성 여부를 이진 분류로 판단합니다. 특히 짧은 펩타이드에 대한 독성 예측에서 기존 1D 서열 기반 방법보다 더 우수한 독성 판별 성능을 갖습니다.

## 작동 원리
- 입력받은 펩타이드 서열에 대하여 ProtT5 (언어모델)를 활용하여 서열 임베딩을 추출합니다.
- ESMfold를 통해서 예측된 구조는 그래프에 임베딩 되고, 이후 GNN (EGNN)을 통하여 3D 기하학적 정보에 대한 학습이 진행됩니다.
- 서열 임베딩과 구조 임베딩을 통합 벡터로 결합한 후 독성/비독성 라벨이 있는 공개된 펩타이드 데이터셋을 모아 통합 학습을 진행하였습니다.

## 입력・출력・설정 옵션

**입력** : 펩타이드 서열 (FASTA 포맷)

**출력** : 독성 여부 이진 분류 결과 (독성, 비독성),  모델 confidence (%)

**설정 옵션** : 사용자 설정 옵션 없음

## 용도・차별 포인트

**용도** :
- 펩타이드 기반 치료제 개발 초기 단계에서 독성 스크리닝을 대규모로 할 때 사용할 수 있습니다.

**차별 포인트** :
- 기존의 1D 서열 기반 독성 예측 모델이 예측하기 힘든 3D 공간적 상호작용을 반영하여 예측할 수 있어서 더 높은 정확성을 갖습니다.
- 특히 짧은 펩타이드 (>20 aa) 에서 기존 모델에 비하여 향상된 성능을 갖는다는 것이 검증되었습니다. (pLDDT>60)인 구조에서는 성능이 더욱 개선됨이 확인되었구요.

## 비교해 볼만한 모델
- **ToxinPred3.0**은 대표적인 1D 서열 기반 펩타이드 독성 예측 모델입니다. 속도는 빠르지만 정확도는 조금 떨어질 수 있어서 ToxinPred3.0로 빠르게 스크리닝을 진행한 후 StructoxNet로 후속 검증을 하는 방식으로 비교해볼 수 있습니다.

## 연계해 볼 만한 모델

- ESMfold보다 더 좋은 pLDDT를 출력해주는 구조 예측 모델이 있다면 (예를 들어 Boltz-2) 해당 모듈을 이용하여 구조를 예측한 후 해당 구조를 바탕으로 예측 성능을 비교해볼 수 있습니다.
- 서열을 재설계해주는 모듈과 연동하여 독성이 적은 서열로 재설계 하는 것도 시도해볼 수 있습니다.
