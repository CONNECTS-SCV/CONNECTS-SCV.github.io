## ToxinPred 3.0 통합

펩타이드 기반 약물의 독성을 예측하는 ToxinPred 3.0이 통합되었습니다.

### 새로운 예측 모드

- **세포독성 예측**: 암세포, 정상세포에 대한 독성
- **혈액독성 예측**: 용혈 위험성 평가
- **면역독성 예측**: 면역계 영향 분석

### 성능 지표

| 항목 | 정확도 |
|------|-------|
| 세포독성 | 94.2% |
| 혈액독성 | 91.8% |
| 면역독성 | 93.5% |

### API 업데이트

```python
from connects import ToxinPred

predictor = ToxinPred(version="3.0")
result = predictor.predict(sequence="PEPTIDE_SEQUENCE")
```
