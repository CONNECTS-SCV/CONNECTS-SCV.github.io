## AlphaFold 3 통합

CONNECTS 플랫폼에 AlphaFold 3가 성공적으로 통합되었습니다.

### 주요 기능

- 단백질-리간드 복합체 예측
- 단백질-DNA/RNA 상호작용 모델링
- 향상된 예측 정확도 (pLDDT > 90)
- 대규모 복합체 분석 가능

### 성능 개선

- 예측 속도 3배 향상
- GPU 활용 최적화
- 배치 처리 지원

### 사용 방법

1. Analysis > Protein Structure 메뉴 선택
2. AlphaFold 3 옵션 선택
3. 서열 또는 PDB 파일 업로드
4. 분석 실행

자세한 내용은 [문서](/docs/alphafold3)를 참조하세요.
