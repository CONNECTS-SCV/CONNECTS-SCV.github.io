## 2025년 1월 서버 정기 점검

안정적인 서비스 제공을 위한 서버 정기 점검을 실시합니다.

### 점검 일정

- **일시**: 2025년 1월 25일(토) 02:00 - 06:00 KST (4시간)
- **영향 범위**: 전체 서비스

### 점검 내용

1. 서버 하드웨어 업그레이드
2. 네트워크 장비 펌웨어 업데이트
3. 보안 패치 적용
4. 데이터베이스 최적화

### 서비스 영향

- 웹 플랫폼 접속
- API 호출
- 분석 작업 실행

진행 중인 분석은 점검 후 자동으로 재개됩니다.

불편을 드려 죄송합니다. 더 나은 서비스로 보답하겠습니다.
